About item banc

item banc


an item banc stores prices of items that are basic human needs.
these information deposits are made in real time by real people
and stored in a decentralized database.
Item Banc Engine takes these deposits from around the world
and uses them together to calculate a new kind of Value base,
similar to purchasing power parity,
where the same basic set of items are compared
around the world to discover
the relative values of currencies.
This new Value base is the Item Banc Index.


The people that enter (deposit) the prices we call Validators.
The Validators are rewarded with special tokens.
These tokens can be used to see prices deposited by other Validators.


The tokens also pay for withdrawing Item Banc Index information as well.
The Engine indexes the Values of currencies by comparing their
relative ability to purchase basic human needs.


There are about 100 or so basic items that we plan to
represent in the Item Banc Index,
and 180 world currencies.
In our initial roll-out, we will focus on 14 items in five categories.
